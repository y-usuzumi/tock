# tock

*tock*是一个目录生成工具。当前仅支持生成markdown目录。

## 用法

### 

```
Usage: tock dir [-x]
  Generate TOC

Available options:
  dir                      The root directory to look for contents
  -x                       Exclude directories that do not have files
                           (recursively)
  -h,--help                Show this help text
```
